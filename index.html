Tactic

TACTIC (Tangible and Tabletop Continuous Interaction) sends touch, tangible and gesture information to your HTML client applications

View project on GitHub

What is it?

TACTIC is Javascript API built on top of TUIO. It translates touch and tangible information into events and behaviours for your html applications. It doest the heavy lifting for you, removing the need for back-end data processing. Just register which events you want to receive and you're good to go.

Overview

TACTIC was designed to work on traditional multi-touch set-ups while also allowing interactions above the table, through gesture detection. This paves the way for new interactions thanks to data merging of different modalities.
